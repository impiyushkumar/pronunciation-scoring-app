# 🎯 PronounceAI

**AI-powered English pronunciation scoring with word-level feedback**

[![Live Demo](https://img.shields.io/badge/🌐_Live_Demo-pronounceai--app.vercel.app-8b5cf6?style=for-the-badge)](https://pronounceai-app.vercel.app)

Upload a 30–45 second English speech recording → get an instant pronunciation score (0–100) with every word color-coded by accuracy, hover tooltips showing confidence & phonemes, and detailed coaching tips generated by Gemini Flash.

**No accounts. No audio stored. No tracking.**

## ✨ How It Works

```
Audio Upload → Validate → Whisper STT → G2P Phonemes → Gemini LLM → Score
                ↓              ↓              ↓              ↓          ↓
          magic bytes    word-level      ARPAbet        error      weighted
          + decode +     timestamps     per word      types +     0–100
          duration       + confidence                 coaching
```

1. **Upload** any audio file (MP3, WAV, M4A, OGG, FLAC) — 30 to 45 seconds of English speech
2. **Validation** runs both client-side (instant UX) and server-side (security) — magic bytes, real decode, duration from frames
3. **Whisper** transcribes with per-word confidence scores (0.0–1.0) and timestamps
4. **G2P** maps each word to its expected ARPAbet phonemes using the CMU Pronouncing Dictionary
5. **Gemini 3.5 Flash** classifies errors (mispronunciation / unclear / omission) and generates actionable phonetic coaching
6. **Scorer** computes a weighted 0–100 score from confidence, clarity, fluency, and completeness

## 🔒 DPDP Compliance

| Principle | Implementation |
|-----------|---------------|
| **Consent** | Explicit checkbox required before upload. Button disabled until checked. |
| **Zero storage** | Audio processed in-memory only. `del` after use. Temp files in `finally` blocks. |
| **Data minimization** | No accounts, cookies, analytics, or tracking. |
| **Purpose limitation** | Audio used solely for scoring within one HTTP request. |
| **Data residency** | Audio stays on Railway server. Only text (transcript + scores) goes to Gemini. |
| **Right to erasure** | Nothing to erase — no data persists after the response. |

## 📊 Scoring Formula

```
score = 0.50 × word_confidence + 0.25 × clarity + 0.15 × fluency + 0.10 × completeness
```

| Component | Weight | Source |
|-----------|--------|--------|
| Word confidence | 50% | Average Whisper `word.probability` |
| Clarity | 25% | % of words above 0.70 confidence |
| Fluency | 15% | Inter-word gap analysis from timestamps |
| Completeness | 10% | Word count vs. expected ~2 words/sec |

Error types are classified by **Gemini 3.5 Flash** (with heuristic fallback): mispronunciation, unclear, or omission — each with a phonetic explanation.

## 🚀 Quick Start

### Prerequisites
- Python 3.11+ with pip
- Node.js 18+ with npm
- ffmpeg (`brew install ffmpeg` / `apt install ffmpeg` / `choco install ffmpeg`)

### Backend
```bash
cd backend
pip install -r requirements.txt
# Optional: set GEMINI_API_KEY for LLM-enhanced feedback
# export GEMINI_API_KEY=your-key-here
uvicorn app.main:app --reload --port 8000
```

### Frontend
```bash
cd frontend
npm install
echo "NEXT_PUBLIC_API_URL=http://localhost:8000" > .env.local
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## ⚖️ Key Trade-offs

| Decision | Trade-off | Why |
|----------|-----------|-----|
| Whisper `tiny` model | Less accurate on accented speech | Fits in ~300MB RAM; LLM compensates with linguistic context |
| No forced alignment | Can't score individual phonemes | Adds ~10s latency + significant compute; word-level confidence is sufficient |
| LLM as fallback | Works without API key (heuristic mode) | App never breaks if Gemini is down or quota exhausted |
| In-memory only | One request at a time | Avoids Redis/Celery complexity for a demo; rate limiter prevents overload |
| No user accounts | Can't track improvement | Minimizes PII surface; intentional for DPDP compliance |
